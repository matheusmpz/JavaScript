Estrutura Condicional

Crie um programa em JavaScript que solicita ao usuário dois números e uma operação aritmética (soma, subtração, multiplicação ou divisão). O programa deve realizar a operação solicitada e exibir o resultado, utilizando estruturas condicionais para determinar qual operação executar.

Possiveis Operações Aritméticas:

1- Adição
2- Subtração
3- Multiplicação
4- Divisão
